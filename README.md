# _Sudoku Solver_

This is a sudoku solver program in C++
___

## :small_blue_diamond: 1. Depedencies
 - C++
 - CMake
 - GTest
 - Qt

___

## :mage_man: 2. How to build
### 2.1. :alembic: Tests
```
mkdir build && cd build
cmake ../tests/
make -j$(nprocs)
```
