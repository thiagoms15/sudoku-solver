# Sudoku solver

<img alt="License" src="https://img.shields.io/static/v1?label=license&message=MIT&color=E51C44&labelColor=0A1033">

This is a sudoku solver program wrote in C++
___

## Depedencies
 - C++
 - CMake
 - GTest
 - Qt

___

## Tests
### How to build
```
mkdir build && cd build
cmake ../tests/
make -j$(nprocs)
```
